  • D-7000, a member of Foundation personnel augmented with comprehensive static immortality;
  • One low-grade computer terminal, upon which this file is stored.

D-7000 is to repeatedly generate randomized universes (and maintain related equipment as necessary) until, if at all possible, a variation of the Prime-Timeline is generated with immunity to SCP-7000's effect.
